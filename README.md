# 🤖 GELGOOG AUTO-FARMING SYSTEM v2.0

บอทฟาร์มอัตโนมัติสุดอัจฉริยะที่ผสานพลังระหว่าง Computer Vision และ Deep Learning (CNN) ออกแบบมาเพื่อการฟาร์มที่ยืดหยุ่น ปลอดภัย และเลียนแบบพฤติกรรมมนุษย์ขั้นสุด

## ✨ ไฮไลท์ฟีเจอร์ (Key Features)

* **🧠 AI Mode (Deep Learning):** บังคับตัวละครหลบหลีกสิ่งกีดขวางแบบ Real-time ด้วยโมเดล CNN (`bot_brain_v2.h5`)
* **🎁 AFK 1 Box Mode:** โหมดประหยัดทรัพยากร (ไม่รัน GPU) ปล่อยตัวละครวิ่งจนกว่าจะเจอกล่องสมบัติแล้วสับสวิตช์ออกเกมทันที
* **🛡️ Anti-AFK Bypass:** ระบบแก้ปริศนาต้านบอท (สุ่มเลือกการ์ด) อัตโนมัติ ด้วยสมการคณิตศาสตร์ NCC (Normalized Cross-Correlation)
* **🏺 Auto Claim Relic:** ระบบสแกนหาไอคอนสมุด Relic (7/7) และดำเนินการกดรับสุ่มของรางวัลให้เองก่อนเริ่มรอบใหม่
* **👻 Human-Like Behavior:** ระบบป้องกันการถูกแบนด้วยการสุ่มพิกัดการคลิกเมาส์, สุ่มความเร็วในการตอบสนอง และมี **Fatigue System** (ระบบจำลองความเมื่อยล้า พักเบรกเองเมื่อรันไปสักระยะ)
* **🖥️ Interactive Dashboard:** หน้าต่าง Terminal UI สำหรับตั้งค่าการเปิด-ปิดฟังก์ชันทั้งหมดก่อนรัน พร้อมบันทึกค่าลง `config.json` อัตโนมัติ

## 📂 โครงสร้างโฟลเดอร์ (Directory Structure)

เพื่อให้บอททำงานได้อย่างสมบูรณ์ กรุณาจัดเตรียมโฟลเดอร์และไฟล์ให้อยู่ในรูปแบบนี้:

```text
gelgoog_v2/
│
├── main.py                 # ไฟล์หลักสำหรับรันโปรแกรม (Dashboard & Loop Control)
├── config.json             # ไฟล์บันทึกการตั้งค่า (สร้างให้อัตโนมัติเมื่อรันครั้งแรก)
│
├── src/                    # โฟลเดอร์เก็บโค้ดระบบหลัก
│   ├── phase1_prep.py      # ระบบจัดการหน้าต่าง, ซื้อไอเทม, และเคลียร์ Relic
│   ├── phase2_gameplay.py  # ระบบ AI ควบคุมตัวละคร / โหมด AFK
│   ├── phase3_endgame.py   # ระบบออกจากด่าน, แก้ Anti-AFK, และกลับหน้า Main
│   └── utils.py            # ฟังก์ชันช่วยเหลือ (คลิกเมาส์, บันทึก Log)
│
├── assets/                 # โฟลเดอร์เก็บรูปภาพ Template Matching
│   ├── ui/                 # 👈 ใส่รูปปุ่มกด, หน้าต่างเมนู, และไอคอนระบบ (เช่น box_icon.png, relic_full_icon.png)
│   └── items/              # 👈 ใส่รูปไอเทมบูสเตอร์ต่างๆ ในเกม
│
├── models/                 
│   └── bot_brain_v2.h5     # ไฟล์โมเดล AI (CNN)
│
└── logs/                   # โฟลเดอร์เก็บประวัติการทำงาน (สร้างให้อัตโนมัติ)
